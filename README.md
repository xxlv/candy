# 如何开始?

你需要clone本项目到你的机器上,执行以下命令:

```
    git clone git@gitlab.dxy.net:xxlv/dxy-robot.git
    cd dxy-robot
    npm install

```
# 怎么用?

scripts里是所有的脚本，自动发送周报什么的。去看看哪里吧。


# 已收录脚本

- 发送周报 (__dxy_weekly.coffee__)
- virtual station项目发布助手（包括一些常见的发布，清时间戳等）(__dxy_virtual-station.coffee__)
